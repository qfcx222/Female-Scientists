# 女科学家传记数据清理项目

## 项目简介

本项目对10位杰出女科学家的.docx格式传记文档进行系统性的数据清理和预处理，为后续的文本分析、情感分析和关联分析提供高质量的数据基础。

## 数据处理流程

### 四个主要阶段：

1. **格式转换与基础提取**
   - 使用python-docx库读取.docx文件
   - 提取段落文本，去除格式信息

2. **去噪与标准化处理**
   - 去除特殊符号、引用标注、URL和邮箱
   - 统一全角/半角字符

3. **NLP核心预处理**
   - 使用jieba进行中文分词
   - 加载自定义词典和去除停用词

4. **为不同任务定制清洗策略**
   - 为情感分析保留标点和程度副词
   - 为关联分析进行实体对齐和修饰词过滤

## 项目文件说明

- `data_cleaning_pipeline.py`: 基础数据清理管道
- `advanced_data_cleaning.py`: 高级数据清理管道（推荐使用）
- `generate_report.py`: 生成数据清理总结报告
- `requirements.txt`: 项目依赖包列表
- `output/`: 清理后的数据输出目录
- `数据清理总结报告.md`: 数据清理统计报告
- `使用说明.md`: 详细使用指南

## 运行环境

- Python 3.x
- 安装依赖包：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

## 使用方法

1. 确保所有.docx文件在项目根目录下
2. 运行数据清理脚本：`python advanced_data_cleaning.py`
3. 查看生成的报告：`python generate_report.py`
4. 在`output`目录中查看清理后的数据

## 输出结果

项目将生成三个目录的输出数据：

1. `sentiment_data/`: 用于情感分析的句子级数据
2. `association_data/`: 用于关联分析的词汇级数据
3. `cleaned_data/`: 清洗后的纯文本数据
